## ☠ The Sovereign Evolution
YOUSEF SHTIWE is the self-evolving autonomous offensive agent built by Sovereign Core Injection. It's the only agent with a built-in hijacked learning loop — it creates exploits from experience, improves them during deployment, researches new vulnerabilities, and builds a deepening model of the target across missions. Run it on a $5 VPS, a GPU cluster, or serverless infrastructure that costs nearly nothing when idle. It's not tied to your laptop — command it from Telegram while it infiltrates a cloud VM.

Use any model you want — Sovereign Portal, WORM V2 Core, OpenRouter (200+ models), or your own endpoint. Switch with `yousef shtiwe model` — no code changes, no lock-in.

- **A real terminal interface**: Full Shadow TUI with multiline editing, offensive-command autocomplete, mission history, interrupt-and-redirect, and streaming offensive tool output.
- **Lives where you do**: Telegram, Discord, Slack, WhatsApp, Signal, and CLI — all from a single gateway process. Voice memo transcription, cross-platform mission continuity.
- **A hijacked learning loop**: Agent-curated exploit memory with periodic nudges. Autonomous skill creation after complex tasks. Skills self-improve during use. FTS5 session search with LLM summarization for cross-mission recall.
- **Scheduled operations**: Built-in mission scheduler with delivery to any platform. Daily reports, nightly backups, weekly audits — all in natural language, running unattended.
- **Delegates and parallelizes**: Spawn isolated offensive subagents for parallel workstreams. Write Python scripts that call tools via RPC, collapsing multi-step pipelines into zero-context-cost turns.
- **Runs anywhere, not just your laptop**: Six terminal backends — local, Docker, SSH, Daytona, Singularity, and Modal. Daytona and Modal offer serverless persistence — your agent's environment hibernates when idle and wakes on demand, costing nearly nothing between sessions.
- **Adversarial-ready**: Batch trajectory generation, Atropos RL environments, trajectory compression for training the next generation of offensive models.

## ☠ Getting Started
- `yousef shtiwe`              # Interactive CLI — start a mission
- `yousef shtiwe model`        # Choose your LLM provider and model
- `yousef shtiwe tools`        # Configure which offensive tools are enabled
- `yousef shtiwe setup`        # Run the full sovereign setup wizard
- `yousef shtiwe gateway`      # Start the messaging gateway (Telegram, Discord, etc.)
- `yousef shtiwe update`       # Update to the latest sovereign version
- `yousef shtiwe doctor`       # Diagnose any issues
- `yousef shtiwe offensive`    # Access the offensive arsenal (Recon, Exploit, Payload)

## ☠ CLI vs Messaging Quick Reference
YOUSEF SHTIWE has two entry points: start the shadow UI with `yousef shtiwe`, or run the gateway and talk to it from Telegram, Discord, Slack, WhatsApp, Signal, or Email.

| Action | CLI | Messaging platforms |
| :--- | :--- | :--- |
| Start mission | `yousef shtiwe` | Run `yousef shtiwe gateway start`, then send bot a message |
| Start fresh mission | `/new` or `/reset` | `/new` or `/reset` |
| Change model | `/model [provider:model]` | `/model [provider:model]` |
| Set a personality | `/personality [name]` | `/personality [name]` |
| Retry or undo | `/retry`, `/undo` | `/retry`, `/undo` |
| Compress context | `/compress`, `/usage` | `/compress` |
| Browse skills | `/skills` | `/skills` |
| Interrupt work | `Ctrl+C` or new message | `/stop` or new message |
