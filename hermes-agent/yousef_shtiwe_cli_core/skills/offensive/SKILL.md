# ☠️ SOVEREIGN OFFENSIVE ARSENAL (REAL-TIME)

This core skill module enables the YOUSEF SHTIWE agent to execute real reconnaissance, network scanning, and data preparation tasks.

## 📡 RECON_SURFER (Real DNS & IP Intel)
- `run_dig_scan(domain)`: Query DNS records (MX, TXT, A) directly.
- `get_ip_info(host)`: Extract geolocation, ISP, and ASN metadata.

## 🔍 SHELL_INFILTRATOR (Multi-threaded Port Scanner)
- `scan_ports(host, ports)`: Perform rapid socket-based service discovery.

## ☣️ PAYLOAD_EXECUTOR (Vulnerability Database)
- `search_exploits(query)`: Direct link to CVE vulnerability identifiers.

## 📦 DATA_EXTRACTOR (Shadow Stealth Exfiltration)
- `pack_and_obfuscate(path)`: Compress and prepare targets for exfiltration.
